# shopify-demo-app-node-react

This repository contains the completed app and starter files for the [Build a Shopify app with Node and React](https://developers.shopify.com/tutorials/build-a-shopify-app-with-node-and-react) tutorial.

Each step of the [Build a Shopify app with Node and React](https://developers.shopify.com/tutorials/build-a-shopify-app-with-node-and-react) tutorial has a corresponding branch in this repository. You are currently on the `embed-your-app-in-shopify-starter-files` branch. Please refer to the [`master`](https://github.com/Shopify/shopify-demo-app-node-react/tree/master) branch to see all the steps.

## Using Embed your app in Shopify starter files

You can use these starter files to skip to the start of [Embed your app in Shopify](https://developers.shopify.com/tutorials/build-a-shopify-app-with-node-and-react/embed-your-app-in-shopify), but make sure that you first complete these setup instructions:

1. [Install the latest stable version of Node.js.](https://developers.shopify.com/tutorials/build-a-shopify-app-with-node-and-react/set-up-your-app#install-the-latest-stable-version)
2. Install npm packages (run: npm install).

Please create an issue if you have any questions or comments.
